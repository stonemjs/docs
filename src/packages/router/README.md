---
title: Stone Router
---

Stone Router etant standalone et independant des plateformes, permet entre autres de definir des routes vers les fonctionnalités d'une application.

Il expose une API simple et efficace incluant:

- Définition de routes explicites
- Définition de routes implicites
- Definition de routes avec decorateurs
- Definition de routes imbriquées
- Parametres pour les routes
- Alias pour les routes
- Configuration globale des routes
- Un systeme de middleware pour intercepter les requetes vers les routes
- Liaison implicite des models depuis les noms de parametre de route
- Evenements pour suivre le flow d'execution
- Personnalisation et extention des fonctionnalités
